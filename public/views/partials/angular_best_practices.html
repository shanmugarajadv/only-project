AngularJS - Best Practices
1. Directory Structure

Create the directory structure as suggested below in the web server root directory:

Use style/ directory to maintain all stylesheet files

Use images/ directory to maintain all static image files

Use scripts/ directory to maintain all javascript source code files

Use scripts/vendor/ directory to maintain all JS library files

Use scripts/controller/ directory to maintain all AngularJS controller files

Use scripts/directives/ directory to maintain all AngularJS custom directive files

Use scripts/services/ directory to maintain all AngularJS custom service files

Use views/ directory to maintain all angular HTML template files

2. Make the page HTML5 Valid

Prepend “data-“ to all angular directives to make it HTML5 valid.

<html data-ng-app="ApplicationName"></html>

In the above example "data-" is appended to angular "ng-app" directive.

3. Declaration of "data-ng-app" attribute

Declare “data-ng-app” as <html> element or to a <div> element to declare the scope of the angular application.

<html data-ng-app="ApplicationName"></html>

In the above example "data-ng-app" is appended to <html> element.

<div data-ng-app="ApplicationName"></div>

In the above example "data-ng-app" is appended to <div> element.

4. Use custom directives to route header.html and footer.html 
                            <body>
                                <my-header></my-header>
                                <div data-ng-view></div>
                                <my-footer></my-footer>
                            </body>
                        
                            sample_app .directive('myHeader', function() {
                                return {
                                    restrict: 'AE',
                                    templateUrl:'templates/header.html'
                                }
                            });

                            sample_app .directive('myFooter', function() {
                                return {
                                    restrict: 'AE',
                                    templateUrl:'templates/footer.html'
                                }
                            });
